IDP Design documentation
========================

**idp-design** is a Python library for simulating and optimizing intrinsically disordered proteins (IDPs) via a continuous sequence representation.
It uses JAX both for molecular dynamics simulations and for automatic differentiation.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project.
